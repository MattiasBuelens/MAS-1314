DYNAMIC PICKUP AND DELIVERY PROBLEMS WITH TIME WINDOWS
======================================================

Mattias Buelens & Daan Gheysens 
Multi-Agent Systems (2014)

Building
========

This project uses Maven for building.

Running
=======

The main program is in the mas.PDP class.
All experiment parameters can be configured within the run() method of this class.
See also mas.SimulationSettings and mas.experiment.Experiment for more information about these parameters.
